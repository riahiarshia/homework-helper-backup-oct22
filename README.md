## 🚀 QUICK START

### Backend Setup:
```bash
cd backend
npm install
cp .env.example .env  # Configure environment
npm start
```

### iOS App:
```bash
cd ios-app/HomeworkHelper
open HomeworkHelper.xcodeproj
# Build and run in Xcode
```

### Run Automated Tests:
```bash
cd backend/tests
./run-automated-tests.sh
```
